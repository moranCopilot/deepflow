# 预生成音频文件目录

此目录用于存放首次生成 FlowList 时的三个预生成音频文件。

## 文件列表

1. **睡前冥想** (`sleep-meditation.mp3` 或 `.m4a`)
   - 场景标签：`sleep_meditation`
   - 时长：约 10 分钟
   - 内容：放松身心，准备入睡的冥想音频

2. **听首歌放松一下** (`relax-music.mp3` 或 `.m4a`)
   - 场景标签：`sleep_meditation`
   - 时长：约 5 分钟
   - 内容：轻松音乐，放松心情

3. **在家充电：科技时事** (`tech-news.mp3` 或 `.m4a`)
   - 场景标签：`home_charge`
   - 时长：约 15 分钟
   - 内容：了解最新科技动态

## 文件格式

推荐使用 MP3 或 M4A 格式。文件名可以自定义，但需要在代码中更新对应的路径引用。

## 注意事项

- 这些文件只在用户首次生成 FlowList 时自动添加到列表中
- 之后即使添加新文件，也不会再次添加这些预生成文件
- 预生成文件不会被删除，会一直保留在 FlowList 中
